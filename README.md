# AI in US Military Support

This repository contains my dataset and analysis code exploring which AI companies are prominent in US military support during recent operations in Iran and Gaza.

## Contents
- `ai-in-us-israel-military-support.ipynb` - Python EDA code
- `Dataset and SupportFile/` - Excel dataset and source links

## Dataset
I compiled this dataset manually from public reports and news sources. All sources are linked in the support files and were verified multiple times using Grok AI for accuracy.

## Objective
Identify and analyze which AI/defense companies appear in public sources as supporting recent US military operations in Iran and Gaza.
